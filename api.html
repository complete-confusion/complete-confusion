complete_confusion.evaluate.save_performance_metrics_to_html(predictions, truths, classes=None, text_representations=None, image_representations=None, output_path='.', probabilities=None)[source]

Creates a confusion matrix and exports it to an HTML file.

Parameters:

  • predictions (Collection[int]) – The predicted class labels.

  • truths (Collection[int]) – The true class labels.

  • text_representations (Optional[Collection[str]]) – Optional text representations of the instances.

  • image_representations (Optional[Collection[str]]) – Optional list of URIs, one URI for each data instance, depicting, summarizing or

  • instance. (representing the data)

  • classes (Optional[Collection[str]]) – List of class names corresponding to labels.

  • probabilities (Optional[Collection[float]]) – The predicted probabilities for each class.

  • output_path (Union[str, Path]) – Path to a folder to save the output files to.
